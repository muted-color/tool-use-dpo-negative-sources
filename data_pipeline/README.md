# Data Pipeline Notes

This directory documents the negative construction and filtering pipeline. Full
raw pair-generation code and full pair JSONL artifacts are not included in this
public repository because the underlying prompt sources are release-restricted or
benchmark-sensitive.

The public release provides:

- source and filtering descriptions in `docs/negative_construction.md`
- audit rubric in `docs/audit_rubric.md`
- aggregate clean/filtering outcomes in `artifacts/`
- sanitized per-example evaluation outputs in `results/per_example/`
