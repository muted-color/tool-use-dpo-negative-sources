# Data Availability Statement

Code, sanitized evaluation outputs, aggregate metrics, bootstrap artifacts,
figure-generation scripts, run manifests, and audit summaries are included in
this repository.

Raw benchmark data, full training examples, full processed DPO pairs, private
audit key files, generated text from benchmark prompts, and model adapters are
not redistributed in this version because of licensing, benchmark-integrity,
organizational, size, or provider-policy constraints.

The released artifacts are intended to support verification of the reported
tables, confidence intervals, and Pareto figures. They are not a complete
retraining bundle.
